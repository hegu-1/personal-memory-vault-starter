# 我对时间管理的看法

## 核心 framing

时间管理不是"塞满每分钟",是"识别哪些时段精力高 → 把高 leverage 任务放进去"。

时间是 fixed,精力 / 注意力是变量。真正的杠杆在"匹配任务到精力状态",不是"挤出更多时间"。

## 三层模型

1. **物理层**:身体 / 睡眠 / 运动 / 饮食 — 决定精力上限
2. **认知层**:深度思考 vs 浅层执行 vs 创意发散,需要不同心理状态
3. **关系层**:跟别人协作 vs 独处时间,各有需求

## 我的实操原则

- 早 9-12 = 深度思考时段,不约会、不开会
- 下午 = 协作 / 会议 / 沟通
- 晚 = 浅层执行(回邮件 / 整理 / 阅读)

具体技巧见 [[example-pomodoro]] (番茄钟) 和 [[example-weekly-review]] (周回顾)。

## 反例 / 不做的事

- 强行 6 点起床(精力不在那)
- 多线程切 task(损耗大)
- 用 to-do list 衡量产出(衡量错维度)
