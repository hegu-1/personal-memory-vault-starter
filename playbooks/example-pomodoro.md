# Pomodoro focus blocks

A concrete technique sitting under the framework in [[example-time-management]].

## The loop

1. Pick one task. Just one.
2. 25 minutes, timer on, single-task. No tab-switching.
3. 5-minute break. Stand up, look away from the screen.
4. After 4 rounds, take a longer 20–30 minute break.

## When I use it

- Deep-work windows (mornings, per [[example-time-management]])
- Tasks I've been avoiding — the 25-minute box lowers the activation cost

## When I don't

- Genuinely creative / divergent work that needs to run long — a hard 25-minute cut breaks flow
- Meetings and collaboration time
