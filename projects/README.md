# projects/

存进行中 / 长期跟进的项目 / 任务。

## 什么时候用

- 跨多日 / 多周的事情
- 涉及多个 sub-task 或者 milestone
- 想 track 进度

## 命名约定

- `<project-name>.md`
- 例:`learn-japanese.md` / `side-business-handmade-soap.md` / `book-draft-novel.md`

## 建议结构

- Goal / Milestone / Status / Stakeholders / Resources
