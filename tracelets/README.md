# tracelets/

Single high-value calibration / decision moments — one file per moment.

## When to use

- You made a choice you **hesitated over / regret / are glad about** and want to record it
- A **calibration moment** while collaborating (they corrected you / you corrected them)
- The instant you reframed a view
- A mistake + the lesson learned

## Naming convention

- `YYYY-MM-DD-<topic-slug>.md`
- e.g. `2026-05-26-decided-to-quit-job.md` / `2026-03-15-friend-feedback-i-was-wrong.md`

## Suggested structure (four parts)

1. **Action** — what happened
2. **Why** — why this moment matters
3. **Alt** — other paths that were possible (that you didn't take / didn't happen)
4. **Outcome / Calibration** — what rule it produced (possibly codified into a [[playbook]])

## Relationship to playbooks

- **tracelet**: records the origin moment (the specific scene + how you framed it then)
- **playbook**: distills the reusable, forward-looking rule

The two link to each other — a tracelet marks `codified-as: [[playbook-slug]]`, and the playbook points back to its origin tracelet.
