# tracelets/

存单个高价值校准 / 决定时刻 — 一个 file 一个 moment。

## 什么时候用

- 你做了一个**让你犹豫 / 后悔 / 庆幸**的选择,想记下来
- 一个跟别人协作时的**校准时刻**(他纠正你 / 你纠正他)
- 你 reframe 一个看法的瞬间
- 一个错误 + lesson learned

## 命名约定

- `YYYY-MM-DD-<topic-slug>.md`
- 例:`2026-05-26-decided-to-quit-job.md` / `2026-03-15-friend-feedback-i-was-wrong.md`

## 建议结构(四元)

1. **Action** — 发生了什么
2. **Why** — 为什么这个 moment 重要
3. **Alt** — 还有什么其他可能的路径(我没选 / 没发生)
4. **Outcome / Calibration** — 沉淀出了什么 rule(可能 codify 进 [[playbooks]])

## 跟 playbooks 关系

- **tracelet**:记录 origin moment(具体场景 + 当时 framing)
- **playbook**:沉淀 reusable rule(forward-looking)

两者互相 link — tracelet 标 `codified-as: [[playbook-slug]]`,playbook 标 origin tracelet。
