# 朋友指出我对她有偏见

**Moment**: 2025-12-10 晚上,跟 [朋友] 聊天到一半,她突然说:"你总觉得我做事拖,但你没问过我背后为什么慢。"

---

## Action(发生什么)

那段时间她答应我帮忙做一件事,delay 了 2 周。我有点 frustrated,跟她说话 tone 不太好。

她那天晚上 explicit 说:"你 default 认为我 lazy,但实际上我那两周家里事情多。你没问。"

## Why(为什么这个 moment 重要)

我 default 模式:**用结果归因人 character**(她拖 → 她 lazy)。

她 reframe:**先问 context 再判断 trait**。

这是个普遍 pattern — 我对**很多人**都这么 default。这次她敢说出来是 gift。

## Alt(其他可能路径)

- 她不说(继续误解 + 友谊磨损)
- 我反驳"那是你没说啊"(defensive,关闭 calibration channel)
- 我承认错误但只这一次,模式不变

实际选:**承认 + codify pattern + future 改**。

## Calibration outcome

Codify 进:
- [[example-boundary]] 加一条:对人 attribute trait 前,先问 context

跟 [[example-self]] 的 self-awareness 一脉相承。

**Future self-check**:任何时候我对某人"判定"前,问自己:"我问过他/她背后吗?"
