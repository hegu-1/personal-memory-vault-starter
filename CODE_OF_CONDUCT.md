# Code of Conduct

This project follows a simple principle: **be respectful**.

- Be kind and constructive in issues, pull requests, and discussions.
- Assume good faith. Critique ideas, not people.
- No harassment, discrimination, or personal attacks.
- This is a small public-domain project maintained in spare time — patience is appreciated.

Violations can be reported to the maintainer through GitHub. The maintainer may edit or remove comments, commits, and contributions that violate this code, and may block repeat offenders.

Adapted in spirit from the [Contributor Covenant](https://www.contributor-covenant.org).
