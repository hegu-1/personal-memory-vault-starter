# Changelog

All notable changes to this template are documented here. Loosely follows [Keep a Changelog](https://keepachangelog.com); versions are date-stamped.

## [1.1.0] — 2026-06-14

### Added
- `playbooks/example-vault-maintenance.md` + `playbooks/example-weekly-review.md` — the **operating layer**: how to keep a vault alive (capture / session-start re-entry / weekly consolidation / monthly prune).
- `playbooks/example-pomodoro.md` — resolves the technique links from the time-management concept.
- `scripts/build-index.py` — regenerates `INDEX.md` from node frontmatter, with a `--check` mode for CI / pre-commit.
- README: a "network at a glance" graph of the example nodes, and a "Going further" pointer to the `personal-ai-os` thesis.

### Changed
- `INDEX.md` is now generated by `scripts/build-index.py`.
- Enriched cross-links among the example nodes so they form a coherent, navigable network.
- Softened the references to the companion PDFs.

## [1.0.0] — 2026-06-14

First tagged public release.

### Changed
- Unified documentation language to **English-primary** across all directory READMEs, `INDEX.md`, and every `example-*.md`.
- Kept `HUMAN_GUIDE.md` as the **Chinese beginner on-ramp**, now clearly flagged in `README.md` and at the top of the guide.

### Added
- `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, this `CHANGELOG.md`, and issue / pull-request templates.
- README badges and author credit.
- Repository marked as a GitHub template; topics added.

## [0.1.0] — 2026-05-26

Initial release of the personal-memory-vault-starter: nine node-type directories, example nodes, the `README` / `LLM_GUIDE` / `HUMAN_GUIDE` docs, and the CC0 license.
